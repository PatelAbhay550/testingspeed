# Testing Speed

A fast and reliable speed test application built with Node.js and Express, leveraging the Fast Speedtest API for accurate internet speed measurements.

## Features

- Real-time internet speed testing
- Simple and intuitive interface
- Accurate measurements using Fast Speedtest API

## Technologies Used

- **Runtime**: Node.js
- **Framework**: Express.js
- **Frontend**: HTML, JavaScript
- **Version Control**: Git/GitHub

## Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

## Installation

1. Clone the repository:
```bash
git clone https://github.com/PatelAbhay550/testingspeed.git
cd testingspeed
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```

## Dependencies

- `express`: ^5.2.1
- `fast-speedtest-api`: ^0.3.2
